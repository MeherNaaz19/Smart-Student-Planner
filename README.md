# Smart Student Planner 📅

A multi-module student productivity web app built as a course project using HTML, CSS and JavaScript. Features a clean dark UI with sidebar navigation and real-time interactive modules.

🔗 **Live:** https://mehernaaz19.github.io/Smart-Student-Planner/

---

## 📌 Features

- **Dashboard** — Stats overview (tasks due, study streak, weekly hours, GPA), today's tasks and schedule, upcoming deadlines
- **Assignments** — Track assignments with subject tags, status and due dates
- **Schedule** — Class timetable with time slots, room and professor info
- **Subjects** — Subject-wise progress tracking with grades
- **Focus Timer** — Pomodoro timer for focused study sessions
- **Analytics** — Subject-wise performance bar charts and study hour tracking
- **Settings** — Notification and display preferences
- **Login System** — Student login screen with session management

---

## 🛠️ Tech Stack

- HTML
- CSS (custom dark theme, CSS variables)
- Vanilla JavaScript
